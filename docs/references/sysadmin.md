# Sysadmin References

Flat source catalog for system administration and infrastructure work.

## Tier 0

| Area | Source | URL | Use for |
|---|---|---|---|
| Internet standards | RFC Editor | https://www.rfc-editor.org/ | Published RFCs and protocol terminology. |
| Draft standards | IETF Datatracker | https://datatracker.ietf.org/ | Draft status and working group context. |
| Linux kernel | Linux Kernel Documentation | https://docs.kernel.org/ | Kernel behavior, networking stack, cgroups, filesystems. |
| Linux manuals | Linux man-pages project | https://man7.org/linux/man-pages/ | Syscalls, procfs, capabilities, common Linux interfaces. |
| systemd | systemd documentation | https://systemd.io/ | systemd, journald, networkd, resolved. |
| SSH | OpenSSH manuals | https://www.openssh.com/manual.html | ssh, sshd, client/server config, key handling. |
