# Research Queue

This folder tracks sources that need evaluation before they are allowed into the skills base.

## Workflow

1. Add source to `queue.md`.
2. Run `rules/bullshit-check.md`.
3. Create a source card under `sources/`.
4. Assign authority tier.
5. Extract reusable workflow ideas.
6. Rewrite into a narrow skill only if it passes review.

## Do not skip evaluation

A repository with stars is not automatically trustworthy. A polished README is not proof. An MCP server is not automatically safe. A skill pack is not automatically expertise.

The base must be boring, explicit, and grounded.
