# Source Review: terminal-skills

## Claimed purpose

A collection of terminal-oriented skills / command references for Linux, Docker, Kubernetes, cloud CLIs, databases, and operations tasks.

## Authority level

Cheat sheet only.

## Strengths

- Useful broad index of operational topics.
- Good for command discovery.
- Can reveal common workflow categories that deserve proper skills.

## Weaknesses / caveats

- Breadth over depth.
- Often lacks operational context, risk classification, and validation.
- Command lists are not runbooks.
- May omit safety constraints.

## What can be reused

- Topic list.
- Basic command reminders.
- Candidate skill names.

## What must not be reused blindly

- Production commands.
- Troubleshooting conclusions.
- Security posture.
- Any command that mutates state.

## Verdict

Useful only as a map of possible command areas. Rewrite into proper skills with authority chain, forbidden actions, dry-run checks, and approval gates.
